# AIEXPO WEB

AI 기반 여행지 추천 및 여행 커뮤니티 웹 애플리케이션

## 📌 프로젝트 소개

AIEXPO는 사용자에게 맞춤형 여행지를 추천하고, 여행 계획을 관리하며, 여행 경험을 공유할 수 있는 종합 여행 플랫폼입니다. AI 기반의 검색 기능과 커뮤니티 기능을 통해 더 나은 여행 경험을 제공합니다.

## ✨ 주요 기능

### 🔐 인증 시스템
- **회원가입**: 이메일 인증을 포함한 4단계 회원가입 프로세스
- **로그인**: 안전한 사용자 인증
- **비밀번호 찾기**: 이메일 인증을 통한 비밀번호 재설정

### 🏠 홈 페이지
- **여행지 추천**: AI 기반 맞춤형 여행지 추천
- **일반 검색**: 키워드 기반 여행지 검색
- **슈퍼 검색**: AI 강화 검색 (크레딧 사용)
- **테마별 필터**: 역사, 자연, 문화 등 다양한 테마 필터
- **북마크**: 관심 여행지 저장 기능

### 🌍 커뮤니티
- **여행지 랭킹**: 인기 여행지 순위
- **블로그**: 사용자 여행기 공유
- **정렬 및 필터**: 다양한 조건으로 콘텐츠 탐색

### 👤 마이페이지
- **프로필 관리**: 사용자 정보 수정
- **크레딧 관리**: 보유 크레딧 확인 및 충전
- **여행 기록**: 나의 여행 기록 관리
- **북마크 목록**: 저장한 여행지 관리
- **계정 설정**: 아이디/비밀번호 변경, 로그아웃, 탈퇴

## 🛠 기술 스택

### Frontend
- **React 19.2.0**: UI 라이브러리
- **React Router DOM 7.13.0**: 라우팅
- **Vite**: 빌드 도구 및 개발 서버
- **CSS Modules**: 스타일링

### Development Tools
- **ESLint**: 코드 품질 관리
- **@vitejs/plugin-react-swc**: React Fast Refresh


## 📱 페이지 구성

| 경로 | 페이지 | 설명 |
|------|--------|------|
| `/` | - | 로그인 페이지로 리다이렉트 |
| `/login` | 로그인 | 사용자 로그인 |
| `/signup` | 회원가입 | 4단계 회원가입 프로세스 |
| `/find-password` | 비밀번호 찾기 | 비밀번호 재설정 |
| `/welcome` | 환영 | 회원가입 완료 |
| `/home` | 홈 | 여행지 추천 및 검색 |
| `/community` | 커뮤니티 | 랭킹 및 블로그 미리보기 |
| `/community/ranking` | 랭킹 상세 | 전체 여행지 랭킹 |
| `/community/blog` | 블로그 상세 | 전체 여행 블로그 |
| `/profile` | 마이페이지 | 사용자 프로필 및 여행 기록 |

## 🚀 배포 설정

Docker 이미지 실행 시 백엔드 업스트림 주소를 지정해야 `/api/*` 요청이 정상 프록시됩니다.

- `BACKEND_UPSTREAM`: 백엔드 서버 주소 (예: `http://backend:8080`, `http://10.0.0.12:8080`)

예시:

```bash
docker run -d \
  -p 80:80 \
  -e BACKEND_UPSTREAM=http://backend:8080 \
  aiexpo-web:latest
```
